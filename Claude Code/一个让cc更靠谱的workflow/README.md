## 说明：
⚡ 核心问题
[1]Claude经常创建重复文件
[2]忽略现有代码结构
[3]胡编乱造不存在的文件
[4]破坏项目架构
	
🔧 解决方案：5步骤强制模板
📋 STEP 1: 强制读取规则文档
🔍 STEP 2: 分析现有系统
🎯 STEP 3: 制定实施计划
🔧 STEP 4: 提供技术细节
✅ STEP 5: 完成交付物
	
💡 关键规则设置
✅ 优先复用现有代码，避免创建新文件
✅ 只能重构，不能重写
✅ 必须引用具体文件路径
✅ 遵循现有架构
❌ 禁止无理由创建新文件
❌ 禁止忽略现有代码库
❌ 禁止泛泛而谈
	
🎯 实际效果
使用这个模板后：
[1]Claude不再胡编文件
[2]不再破坏文件夹结构
[3]收获一个靠谱的同事
